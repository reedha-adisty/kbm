TAHAP TUJUAN PEMBELAJARAN (TP) POKOK BAHASAN KEGIATAN UTAMA HASIL AKHIR
1
2.1.1.
🧩
Menjelaskan konsep algoritma: logis, sistematis, dan terstruktur.
  • Pengertian dan ciri algoritma
  • Berpikir logis dan sistematis
  • Struktur penyusunan algoritma
  • Contoh algoritma dalam kehidupan sehari-hari
  • ✓ Mengamati contoh prosedur sehari-hari
  • ✓ Mengidentifikasi ciri logis dan sistematis
  • ✓ Menyusun langkah penyelesaian masalah sederhana
  • ✓ Mendiskusikan struktur algoritma yang baik
🎯Pemahaman konsep algoritma yang logis, sistematis, dan terstruktur.
2
2.1.2.
📦
Membedakan struktur data sederhana (array, list, stack, queue) dan fungsinya.
  • Array (struktur dan indeks)
  • List (penambahan dan penghapusan data)
  • Stack (LIFO)
  • Queue (FIFO)
  • ✓ Mengenal karakteristik tiap struktur data
  • ✓ Membandingkan cara kerja array, list, stack, queue
  • ✓ Mensimulasikan operasi dasar tiap struktur data
  • ✓ Menentukan struktur data sesuai kasus
🗃️Kemampuan membedakan dan memilih struktur data sesuai kebutuhan.
3
2.1.3.
🧮
Menerapkan proses komputasi: pengumpulan, pengolahan, dan validasi data.
  • Pengumpulan data
  • Pengolahan data (perhitungan, pengurutan)
  • Validasi dan pengecekan data
  • Penyajian hasil komputasi
  • ✓ Mengumpulkan data dari suatu kasus
  • ✓ Mengolah data sesuai kebutuhan
  • ✓ Memvalidasi keakuratan dan kelengkapan data
  • ✓ Menyajikan hasil pengolahan data
📊Data terkumpul, terolah, dan tervalidasi dengan benar.
4
2.1.4.
🔁
Menerapkan algoritma sekuensial, percabangan, dan perulangan untuk masalah sederhana.
  • Algoritma sekuensial
  • Struktur percabangan (if-else)
  • Struktur perulangan (loop)
  • Penerapan pada masalah sederhana
  • ✓ Menyusun langkah algoritma sekuensial
  • ✓ Menerapkan percabangan pada kasus pengambilan keputusan
  • ✓ Menerapkan perulangan pada kasus berulang
  • ✓ Menguji dan memperbaiki algoritma
⚙️Algoritma sekuensial, percabangan, dan perulangan diterapkan dengan tepat.
5
2.1.5.
📝
Merancang solusi dalam bentuk pseudocode yang sistematis.
  • Kaidah penulisan pseudocode
  • Notasi input, proses, output
  • Penerapan struktur kontrol pada pseudocode
  • Evaluasi dan perbaikan pseudocode
  • ✓ Menentukan masalah dan kebutuhan solusi
  • ✓ Menyusun pseudocode secara sistematis
  • ✓ Menerapkan struktur kontrol pada pseudocode
  • ✓ Memeriksa dan menyempurnakan pseudocode
📃Pseudocode solusi yang sistematis dan siap diimplementasikan.
6
2.1.6.
🖥️
Menjelaskan dan mensimulasikan model IPO pada arsitektur Von Neumann.
  • Komponen arsitektur Von Neumann
  • Model Input-Process-Output (IPO)
  • Alur data pada CPU dan memori
  • Simulasi sederhana model IPO
  • ✓ Mengidentifikasi komponen arsitektur Von Neumann
  • ✓ Menjelaskan alur input, proses, dan output
  • ✓ Mensimulasikan model IPO pada kasus sederhana
  • ✓ Mempresentasikan hasil simulasi
💻Simulasi model IPO pada arsitektur Von Neumann dipahami dengan baik.
7
2.1.7.
🔐
Menjelaskan peran sistem operasi serta mengatur privasi dan keamanan akun digital.
  • Fungsi dan peran sistem operasi
  • Manajemen file dan proses pada sistem operasi
  • Pengaturan privasi akun digital
  • Keamanan akun dan perangkat
  • ✓ Menjelaskan fungsi utama sistem operasi
  • ✓ Mengelola file dan proses pada sistem operasi
  • ✓ Mengatur privasi pada akun digital
  • ✓ Menerapkan langkah keamanan akun
🛡️Pemahaman peran sistem operasi serta akun digital yang aman dan terlindungi.
